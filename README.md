## 📚 Curriculum Breakdown

The repository categorizes knowledge into logical blocks. Study these in order to build your skills.

### Mathematics
The math section covers the logic behind AI. It includes guides on matrix operations, calculus for optimization, and statistical inference. You should focus on these before you move to deep learning.

### Computer Vision and NLP
These directories explain how computers "see" images and "read" text. You will find guides on how models turn pixels into numbers, and how machines assign meaning to human language.

Python is the language of modern AI. The JAX folder contains explanations of high-performance libraries. These tools allow computers to do math very quickly.
